# Glossário: Módulo 1 (Fundamentos da Análise de Dados)

Aqui estão as definições oficiais dos principais termos que você aprendeu neste módulo, ideais para revisão rápida:

*   **Habilidades analíticas:** Qualidades e características inatas associadas ao uso de fatos para resolver problemas.
*   **Pensamento analítico:** O processo lógico de identificação e definição de um problema e, em seguida, sua solução por meio do uso de dados de forma organizada e passo a passo.
*   **Contexto:** A condição, o cenário ou o ambiente em que algo existe ou acontece.
*   **Dados:** Uma coleção bruta de fatos (números, palavras, imagens, etc.).
*   **Análise de dados (Data Analysis):** A coleta, a transformação e a organização de dados para tirar conclusões, fazer previsões e orientar a tomada de decisões informadas.
*   **Analista de dados:** Aquele que coleta, transforma e organiza os dados para resolver problemas específicos do negócio.
*   **Análise de dados avançada (Data Analytics):** A ciência ampla dos dados (envolve a gestão de todo o ciclo de vida da informação, desde a coleta até o arquivamento).
*   **Design de dados:** A forma lógica como as informações são estruturadas e organizadas (ex: estruturação de bancos de dados ou planilhas).
*   **Tomada de decisões baseada em dados (Data-Driven Decision-Making):** O uso de fatos reais para guiar a estratégia do negócio, substituindo a intuição ou o "achismo".
*   **Ecossistema de dados:** Os vários elementos (nuvem, servidores, analistas, engenheiros, ferramentas) que interagem entre si para produzir, gerenciar, armazenar, organizar, analisar e compartilhar informações.
*   **Ciência de dados (Data Science):** O campo de estudo que foca no desconhecido. Usa dados brutos e programação complexa para criar novas formas de modelagem e descobrir perguntas que a empresa ainda nem sabia que tinha.
*   **Estratégia de dados:** O gerenciamento das *pessoas*, dos *processos* e das *ferramentas* que são usados em um projeto de análise de dados.
*   **Visualização de dados:** A representação gráfica dos dados (ex: painéis, gráficos, mapas) para facilitar o entendimento de pessoas não técnicas (como diretores).
*   **Conjunto de dados (Dataset):** Uma coleção agrupada e organizada de dados que pode ser manipulada como uma unidade (ex: uma única tabela do banco de dados ou planilha do Excel).
*   **Análise de lacunas (Gap Analysis):** O método usado para examinar onde a empresa está no *presente* e compará-lo com onde ela quer chegar no *futuro*, criando uma "ponte" para a melhoria.
*   **Causa raiz (Root Cause):** O motivo original (o "coração") pelo qual um problema ocorre, indo além dos sintomas superficiais.
*   **Mentalidade técnica:** A capacidade de pegar um problema gigante ou complexo e dividi-lo em etapas menores, trabalhando nelas de forma ordenada e sistemática.
