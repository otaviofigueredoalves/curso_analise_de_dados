# O Papel e a Importância dos Dados

* **Conceito de Dados:** Dados consistem em fatos, como números, palavras e imagens, que as organizações usam para melhorar processos, identificar tendências e tomar decisões.
* **Analista de Dados:** Uma pessoa que coleta, transforma e organiza dados para ajudar na tomada de decisões informadas e baseadas em fatos.

# O que um Analista de Dados faz?

* Ajuda as organizações analisando dados para orientar a tomada de decisão.

# O que vou aprender neste curso?

* **O Processo de Análise de Dados:**
  * PERGUNTAR (Ask)
  * PREPARAR (Prepare)
  * PROCESSAR (Process)
  * ANALISAR (Analyze)
  * COMPARTILHAR (Share)
  * AGIR (Act)
