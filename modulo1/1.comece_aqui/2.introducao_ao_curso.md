# Introdução à Análise de Dados
"Dados, dados, dados. Eu não posso fazer tijolos sem argila"

* Dados são o bloco de construção essencial para a análise e estão no centro das operações de negócios. Eles permitem uma melhor tomada de decisão que pode melhorar as vendas e outros resultados de negócios.
* A análise de dados envolve a coleta de fatos e informações para tirar conclusões, fazer previsões e apoiar a tomada de decisões informadas.

# Plano de Carreira e Oportunidades

* As habilidades de análise de dados são aplicáveis em muitos setores e funções, incluindo finanças, onde os analistas usam dados para orientar as decisões de negócios.
