# Entendendo os Tipos de Dados

Antes de começar a analisar os dados, é útil revisar os quatro tipos de dados mais comuns que você encontrará como analista:

* **Dados Numéricos:** Dados que consistem em números que podem ser medidos, contados ou usados em cálculos matemáticos.
  * Exemplos: Quantidade, preço, altura, idade.

* **Dados de Texto (ou Dados String):** Dados que consistem em letras, palavras, frases ou uma mistura de letras e números usados como um identificador, não para cálculo.
  * Exemplos: Nome do produto, endereço do cliente, anotações, números de identificação (como número de telefone ou CPF/RG, com os quais você não faz cálculos matemáticos).

* **Dados de Data (Date Data):** Dados que representam um ponto específico no tempo, como uma data ou um carimbo de data/hora (timestamp). Esse tipo de dado é crucial para rastrear tendências e eventos ao longo do tempo.
  * Exemplos: Data do pedido, data de envio, data de nascimento.

* **Dados Categóricos:** Dados que podem ser divididos em grupos ou categorias. Esses dados geralmente representam rótulos, nomes ou categorias discretas.
  * Exemplos: Cor do produto (vermelho, azul, verde), status de fidelidade do cliente (Ouro, Prata, Bronze) ou uma avaliação que usa palavras (Excelente, Bom, Ruim).
